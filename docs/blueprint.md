# **App Name**: MindScape

## Core Features:

- Mind Map Generation: Generate a multi-layered mind map from a given topic including topic, sub-topic, category, sub-category and description.
- Card-based Display: Display each node of the mind map as an interactive card with clear hierarchy.
- Layer Navigation: Navigate through the layers of the mind map to view different levels of detail.
- Description Display: Show a brief description for each sub category
- Topic Suggestion: AI-powered tool suggesting the main topic of a mindmap based on previously entered topics or other trending topics

## Style Guidelines:

- Primary color: Soft blue (#94B49F) to evoke calmness and focus.
- Background color: Light beige (#F5F5DC) to provide a non-distracting environment.
- Accent color: Muted green (#90EE90) for interactive elements and highlights, promoting a sense of growth and clarity.
- Font: 'Inter' sans-serif for both headers and body text, ensuring readability and a clean, modern look.
- Card-based layout with clear visual hierarchy and spacing to avoid distraction.
- Use of minimalist icons to represent categories and subcategories to improve comprehension.
- Subtle animations when navigating through layers or expanding cards to provide a smooth user experience.
