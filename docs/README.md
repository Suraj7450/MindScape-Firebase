## Quick Reference

### Key Features
- **AI Mind Mapping**: Generate structured knowledge maps from any topic
- **6 Generation Modes**: Single, Compare, Vision (image/PDF/text), Saved, Self-reference, Regeneration
- **Multi-Language**: 50+ languages with AI translation
- **Nested Exploration**: Infinite depth sub-maps
- **Image Generation**: AI-powered visuals for concepts
- **Community Sharing**: Publish and discover public maps
- **Real-Time Sync**: Firestore listeners for collaborative editing
- **Chat Assistant**: Conversational AI for explanations and quizzes

### Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui (35 components)
- **Backend**: Firebase (Auth, Firestore, Storage)
- **AI**: Google Gemini 2.0 Flash + Pollinations
- **Search**: Google Custom Search API
- **Animations**: Framer Motion

### Firebase Collections
- `users/{uid}` - User profiles and statistics
- `users/{uid}/mindmaps/{mapId}` - Saved mind maps
- `users/{uid}/activities/{activityId}` - Activity tracking
- `publicMindmaps/{mapId}` - Community-shared maps

### API Routes
- `POST /api/generate-image` - AI image generation
- `POST /api/generate-quiz-direct` - Quiz generation
- `GET /api/search-images` - Image search
- `GET /api/test-env` - Environment check

### Server Actions (18+)
- Mind map generation (single, compare, vision, text)
- Node expansion and explanation
- Translation and summarization
- Image prompt enhancement
- Topic recommendations
- Community publishing
